# Phase 1 — UI Design Contract

> Visual and interaction contract for the eSIM Panda design system foundation. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (custom Tailwind v4 + Motion) |
| Preset | not applicable |
| Component library | none (hand-rolled primitives with Tailwind v4 CSS-first config) |
| Icon library | Lucide React 1.8.0 |
| Font | Plus Jakarta Sans (primary), Inter (fallback), system-ui (system fallback) |

**Rationale:** RESEARCH.md recommends Plus Jakarta Sans for its slightly more distinctive/premium feel over Inter. Both are geometric modern families with excellent legibility. Loaded via `next/font/google`.

---

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, tight label spacing |
| sm | 8px | Compact element spacing, icon-to-text gap in nav items |
| md | 16px | Default element spacing, card internal padding, form field gaps |
| lg | 24px | Section padding, card outer margin, bottom nav height padding |
| xl | 32px | Layout gaps between major content blocks |
| 2xl | 48px | Major section breaks on landing page |
| 3xl | 64px | Page-level vertical spacing, hero section padding |

Exceptions:
- Bottom tab bar touch targets: 48px minimum height (meets WCAG 2.5.8 target size)
- Bambu mascot container: 120px default size (not grid-aligned, character sizing)

---

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 16px | 400 (Regular) | 1.5 | Paragraph text, descriptions, form labels |
| Label | 14px | 500 (Medium) | 1.4 | Nav tab labels, badges, metadata, captions |
| Heading | 24px | 700 (Bold) | 1.2 | Page titles, section headings, card titles |
| Display | 36px | 800 (ExtraBold) | 1.1 | Landing page hero headline, marketing callouts |

**Weights used:** 400 (Regular) and 700 (Bold) are the two primary weights. 500 (Medium) is used only for labels/nav. 800 (ExtraBold) is used only for the display/hero size.

**Font loading:** Plus Jakarta Sans via `next/font/google` with `subsets: ['latin']` and `display: 'swap'`. Weights: 400, 500, 700, 800.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF | Page backgrounds, content surfaces, form backgrounds |
| Secondary (30%) | #F5F5F5 | Card backgrounds, nav bar background, section alternation, input backgrounds |
| Accent (10%) | #2979FF | See reserved list below |
| Destructive | #E53935 | Destructive action buttons, error text, Bambu error state accent |

### Accent (#2979FF) Reserved For

- Primary CTA buttons (fill background)
- Active bottom nav tab indicator
- Text links (inline and standalone)
- Toggle/switch active state
- Focus ring on interactive elements
- Progress indicators
- Bambu's accessories (binoculars strap, bamboo highlights)

## Animation Contract

### UI Elements (smooth and elegant)

| Animation | Easing | Duration | Properties |
|-----------|--------|----------|------------|
| Button press | ease-in-out | 150ms | scale: 1 -> 0.97 |
| Card hover shadow | ease-in-out | 200ms | box-shadow transition |
| Focus ring | ease-in-out | 150ms | ring opacity 0 -> 1 |
| Page transition in | ease-in-out | 300ms | opacity: 0->1, x: 20->0 |
| Page transition out | ease-in-out | 300ms | opacity: 1->0, x: 0->-20 |
| Toast enter | ease-out | 200ms | slide from top, opacity |
| Nav indicator slide | spring (stiffness: 400, damping: 25) | auto | x position |
| Theme toggle | ease-in-out | 300ms | rotate 180deg on swap |

### Bambu Animations (playful and bouncy)

| Animation | Type | Config | Notes |
|-----------|------|--------|-------|
| All Bambu motion | spring | stiffness: 300, damping: 15 | Overshoot is intentional |
| Loading loop | spring + keyframes | continuous, interruptible | Eating bamboo cycle |
| Success bounce | spring | stiffness: 400, damping: 10 | Higher energy, more bounce |
| Error wobble | spring | stiffness: 200, damping: 20 | Gentler, sympathetic |
| Idle sway | spring | stiffness: 100, damping: 30 | Very subtle, breathing feel |
| Pose transitions | spring | stiffness: 300, damping: 15 | Cross-fade between SVG poses |

### Haptic Feedback (mobile)

| Action | Pattern |
|--------|---------|
| Button tap | light impact (if navigator.vibrate available) |
| Nav tab switch | light impact |
| Success confirmation | success pattern (two short pulses) |
| Error | warning pattern (one long pulse) |

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA (landing) | "Get Connected" |
| Secondary CTA (landing) | "Browse Plans" |
| Empty state heading (My eSIMs) | "No eSIMs yet" |
| Empty state body (My eSIMs) | "Find the perfect plan for your next destination. Bambu will help you get connected in minutes." |
| Empty state CTA (My eSIMs) | "Browse Destinations" |
| Error state (generic) | "Something went wrong. Try again or contact us on WhatsApp for help." |
| Error state (network) | "Could not connect. Check your internet and try again." |
| Loading state | No text -- Bambu eating bamboo animation only |
| Nav: Home | "Home" |
| Nav: Destinations | "Destinations" |
| Nav: My eSIMs | "My eSIMs" |
| Nav: Profile | "Profile" |
| Theme toggle (to dark) | "Dark mode" |
| Theme toggle (to light) | "Light mode" |

**Tone:** Friendly, concise, action-oriented. Speak like a helpful friend, not a corporation. Match the playful Bambu personality without being childish. All copy goes through i18n translation keys -- no hardcoded strings in JSX.

**Destructive actions in Phase 1:** None. No user data to delete in this foundation phase.

## Layout Contract

### Mobile (< 768px)

| Element | Position | Height | Behavior |
|---------|----------|--------|----------|
| Header | Fixed top | 56px | Logo left, theme toggle right |
| Content | Below header, above nav | calc(100vh - 56px - 64px) | Scrollable, contains PageTransition |
| BottomNav | Fixed bottom | 64px (48px bar + 16px safe area) | 4 equal tabs, active indicator |

### Desktop (>= 768px)

| Element | Position | Width | Behavior |
|---------|----------|-------|----------|
| Header | Fixed top | 100% | Logo left, horizontal nav center, theme toggle right |
| Content | Below header | max-width 1200px, centered | PageTransition wrapper |
| BottomNav | Hidden | -- | Replaced by header horizontal nav |

**Desktop navigation:** Horizontal nav bar integrated into the header (top nav pattern). Tabs from bottom nav become horizontal links in the header center. This is cleaner than a sidebar for a consumer-facing product with only 4 top-level routes.
