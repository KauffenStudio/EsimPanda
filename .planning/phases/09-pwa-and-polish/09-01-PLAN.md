2. Create `public/sw.js` — hand-written service worker with these event handlers:
   - `install`: Cache app shell URLs: ["/", "/manifest.json", "/icon-192x192.png", "/icon-512x512.png"]. Call `self.skipWaiting()`. Cache name: `esim-panda-v1`.
   - `activate`: Delete all caches not matching current CACHE_NAME. Call `self.clients.claim()`.
   - `fetch`: For navigation requests (event.request.mode === 'navigate'), use network-first with cache fallback to '/'. For image/style requests (event.request.destination === 'image' || 'style' || 'script'), use cache-first with network fallback. CRITICAL: Pass through ALL requests containing `_next/` or `api/` or `supabase` or `stripe` — never cache these. Pass through all non-GET requests.
   - `push`: Parse event.data.json(), call self.registration.showNotification with title, body, icon ("/icon-192x192.png"), badge ("/badge.png"), vibrate [100, 50, 100], data (url, type), and actions array.
   - `notificationclick`: Close notification, get url from event.notification.data.url or event.action mapping, call clients.openWindow(url).
   - Add a `message` event listener for 'CACHE_QR' messages — receives { orderId, qrData, setupGuide } and stores in a separate 'esim-qr-data' cache as JSON Response at key `/api/cached-qr/${orderId}`.

3. Create PWA icon placeholders: Generate simple placeholder PNG files for `public/icon-192x192.png`, `public/icon-512x512.png`, and `public/badge.png`. Use a script with Node canvas or create minimal valid PNGs. The icons should be Bambu panda face on #2979FF background. If canvas isn't available, create them as minimal valid PNG files (can be replaced with final art later) — use a simple approach like copying an existing SVG or creating a basic colored square.

4. Update `src/app/layout.tsx`:
   - Add `<meta name="theme-color" content="#2979FF" />` in the head
   - Add an inline `<script>` in the body (BEFORE children) that registers the service worker:
     ```
     if ('serviceWorker' in navigator) {
       navigator.serviceWorker.register('/sw.js', { scope: '/' });
     }
     ```
   - Add the dark mode hydration fix inline script (BEFORE React hydration):
     ```
     try {
       const stored = JSON.parse(localStorage.getItem('esim-panda-theme') || '{}');
       if (stored.state && stored.state.isDark) {
         document.documentElement.classList.add('dark');
         document.documentElement.style.colorScheme = 'dark';
       }
     } catch (e) {}
     ```
   - Use `dangerouslySetInnerHTML` for both scripts
   - Update the `<html>` tag: remove hardcoded `style={{ colorScheme: 'light' }}` (the hydration script handles this dynamically)
