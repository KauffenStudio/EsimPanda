## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |

### New Components

| Component | Location | Description |
|-----------|----------|-------------|
| Breadcrumb | `src/components/seo/breadcrumb.tsx` | `Home > Destinations > {Country}` with JSON-LD BreadcrumbList. Links use `<Link>` from next-intl. Current segment is plain text (not linked), accent color. Separator: ChevronRight icon from lucide-react at 16px. |
| DestinationHero | `src/components/seo/destination-hero.tsx` | Country flag emoji (48px via 3rem font-size) + country name as Display text. 2-3 sentence intro paragraph below in Body text. Full-width, no background image — clean text-based hero for fast LCP. |
| StudentPickBadge | Inline in PlanCard extension | Badge variant="accent" with star icon. Label: "Student Pick". Applied via plan metadata flag. |

| FAQSection | `src/components/seo/faq-section.tsx` | Accordion-style FAQ. Question text at Heading size (24px/700). Answer text at Body size (16px/400). 3-5 items per destination. ## Structured Data Contract

### Product + Offer (per plan on each destination page)

```json
{
  "@type": "Product",
  "name": "eSIM {country} - {data_gb}GB {duration}",
  "offers": {
    "@type": "Offer",
    "price": "{price}",
    "priceCurrency": "EUR",
    "availability": "https://schema.org/InStock"
  }
}
```

### FAQPage (per destination page)

```json
{
  "@type": "FAQPage",
  "mainEntity": [
    { "@type": "Question", "name": "...", "acceptedAnswer": { "@type": "Answer", "text": "..." } }
  ]
}
```

### BreadcrumbList (per destination page)

```json
{
  "@type": "BreadcrumbList",
  "itemListElement": [
    { "position": 1, "name": "Home", "item": "https://esimpanda.com/{locale}" },
    { "position": 2, "name": "Destinations", "item": "https://esimpanda.com/{locale}/browse" },
    { "position": 3, "name": "{country}" }
  ]
}
```

## Interaction Patterns

| Interaction | Behavior |
|-------------|----------|
| FAQ accordion | Click question to expand/collapse. Only one open at a time. 200ms ease-out transition on height. No animation library needed -- CSS transition on max-height. |
| Language switcher | Click globe+code to open dropdown. Click option to switch locale. URL updates from `/en/esim/portugal` to `/fr/esim/portugal`. Page does not full-reload (next-intl router navigation). Dropdown closes on selection or outside click. |
| Breadcrumb navigation | Each segment is a link except the current page. Hover: underline on link segments. |
| Destination card click | Navigate to `/[locale]/esim/[slug]`. No expand animation. Standard page transition. |
| Plan card click | Navigate to `/[locale]/checkout?plan={id}` (existing behavior). |
