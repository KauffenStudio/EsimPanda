**Admin auth check helper** (defined at top of file):
```typescript
async function isAdmin(): Promise<boolean> {
  if (process.env.NEXT_PUBLIC_STRIPE_MOCK === 'true') return true; // mock mode: always admin
  const { createClient } = await import('@/lib/supabase/server');
  const supabase = await createClient();
  const { data: { user } } = await supabase.auth.getUser();
  return user?.app_metadata?.role === 'admin';
}
```

**GET handler** — List all influencer coupons:
- Check isAdmin(), return 403 with `{ error: 'Forbidden' }` if not admin
- Call `getMockInfluencerCoupons()` from `@/lib/referral/mock`
- Compute summary stats: `{ active_count, total_uses, total_revenue_cents }` by reducing over the array
- Return `{ coupons: InfluencerCoupon[], summary: { active_count, total_uses, total_revenue_cents } }`

**POST handler** — Create new influencer coupon:
- Check isAdmin(), return 403 if not
- Parse body: `{ code: string, influencer_name: string, social_url: string, notes: string }`
- Validate with zod schema:
  ```typescript
  const createSchema = z.object({
    code: z.string().min(3).max(20).transform(s => s.toUpperCase()),
    influencer_name: z.string().min(1).max(100),
    social_url: z.string().url().or(z.literal('')),
    notes: z.string().max(500).default(''),
  });
  ```
- **Uniqueness check:** Normalize code to uppercase. Check against:
  1. Static COUPONS array codes (e.g., STUDENT15)
  2. Existing influencer coupon codes from getMockInfluencerCoupons()
  If code exists in either, return 409 with `{ error: 'code_exists' }`
- Create InfluencerCoupon object with defaults:
  ```typescript
  {
    code: validated.code,
    influencer_name: validated.influencer_name,
    social_url: validated.social_url,
    notes: validated.notes,
    discount_percent: 10,

    total_uses: 0,
    total_revenue_cents: 0,
    last_used: null,
    is_active: true,
    created_at: new Date().toISOString(),
  }
  ```
- Call `addMockInfluencerCoupon(coupon)`
- Return 201 with `{ coupon }`

**PATCH handler** — Update influencer coupon (deactivate/reactivate):
- Check isAdmin(), return 403 if not
- Parse body: `{ code: string, is_active: boolean }`
- Call `updateMockInfluencerCoupon(code, { is_active })`
- Return `{ updated: true }`
